# Security Tool

This python script allows you to check urlscan.io, virustotal, and cisco talos for the reputation of a website. I created this because i wanted a one stop place to query these sites for my job duties (and probably because i'm lazy)

# Assumptions

You need to have api keys for virustotal, urlscan.io, and Cisco Talos for this to work properly. 

#Usage

### Storing your API keys

To securely store api keys, I have provided a .env template file in this repository. You should name each key as follows and insert your keys after them:
    `vt_api_key` and `urlscan_api_key`
